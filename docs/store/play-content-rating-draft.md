# Play Content Rating — taslak cevaplar (ÇözBil)

> Owner Play Console anketini bu taslakla doldurur. Nihai sorumluluk yayıncıda.

## Uygulama özeti

Eğitim: sınav sorusu fotoğrafı → AI adım adım çözüm. Hedef: LGS / YGS / KPSS / Ehliyet.

## Beklenen kategoriler

| Soru alanı | Önerilen cevap | Not |
|------------|----------------|-----|
| Violence | Yok | |
| Sexual content | Yok | |
| Profanity | Yok | |
| Controlled substances | Yok | |
| User-generated content | **Evet (fotoğraf)** | Kullanıcı soru fotoğrafı yükler; moderasyon (SafeSearch) var |
| Social features | Yok / sınırlı | Chat/arkadaş yok |
| Location | Yok | |
| Personal info share | Hesap / fotoğraf işlenir | Privacy + Data Safety ile hizala |
| Age | LGS → 13+ bandı; veli onayı akışı | Kids category **değil** |

## AI transparency

- Çözümler AI üretilir; kullanıcıya kontrol notu gösterilir.
- “%100 doğru” iddiası yok.

## Reklam

MVP: AdMob stub / henüz SDK yok. Ankette “ads” yalnızca gerçek SDK bağlanınca Evet.
